# Java Programming Course - Winter Training 2025

## Course Overview
Comprehensive Java programming course covering core concepts to advanced topics for winter training session.

## Course Structure

### Core Java
- **Day 1**: Introduction to Java, Environment Setup
- **Day 2**: Hello World Program, Data Types
- **Day 3**: Variables, Operators
- **Day 4**: Control Structures (if-else, loops)
- **Day 5**: Methods and Functions
- **Day 6**: Arrays and Strings
- **Day 7**: Object-Oriented Programming Basics
- **Day 8**: Classes and Objects
- **Day 9**: Inheritance and Polymorphism
- **Day 10**: Exception Handling

### Advanced Java
- **Day 11**: Collections Framework
- **Day 12**: Generics and Lambda Expressions
- **Day 13**: File I/O and Serialization
- **Day 14**: Multithreading
- **Day 15**: JDBC and Database Connectivity
- **Day 16**: GUI with Swing/JavaFX
- **Day 17**: Networking and Sockets
- **Day 18**: Design Patterns
- **Day 19**: Testing with JUnit
- **Day 20**: Project Development

## Prerequisites
- Basic computer knowledge
- No prior programming experience required

## Tools Required

- IDE (Eclipse, IntelliJ IDEA, or VS Code)
